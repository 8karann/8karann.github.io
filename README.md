# Karan Bhathal — Marketing Portfolio

Personal portfolio of **Karanveer Singh Bhathal** — growth & digital marketing, AI content
marketing, content strategy and social media management.

**Live case studies:** Webbnett (MBA capstone consultancy) · Namshi (GCC social strategy) ·
Euronet Solar (performance marketing) · PCTE (organic Instagram growth).

Static site — no build step. HTML/CSS/vanilla JS with GSAP, ScrollTrigger and Lenis via CDN.

## Run locally

```
python3 -m http.server 4173
```

then open http://localhost:4173

## Deploy

Any static host works (GitHub Pages, Netlify, Vercel). For GitHub Pages: push to a repo,
then Settings → Pages → Deploy from branch → `main` / root.
